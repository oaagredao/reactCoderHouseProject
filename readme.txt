# Ecommerce de Comida Oriental - Proyecto Ramus

Bienvenido al proyecto Ramus, un simulador de ecommerce de productos de comida oriental tipo ramen.
Este proyecto fue desarrollado utilizando React y varias tecnologías relacionadas.
Te permite explorar una selección de productos de ramen, agregarlos al carrito y realizar 
compras simuladas.

## Características Principales

- Explora una variedad de productos de comida oriental.
- Agrega productos al carrito de compras.
- Realiza compras simuladas utilizando el carrito, agregando y eliminando productos.
- Genera enlaces dinámicos para cada producto utilizando React Router DOM y useParams.
- Utiliza el estado local con `useState` para el manejo de la información del carrito.
- Sincroniza con Firebase Firestore para obtener la información de los productos mediante funciones en `firestore.js`.
- Emplea el contexto con `useContext` para administrar el estado global de la aplicación.
- Utiliza la librería `sweetalert2` para mostrar mensajes y alertas interactivas.
